sdlxx::ttf::SDLXX_ttf::SDLXX_ttf ( )

Construct the SDLXX_ttf object that initializes the font API.

This must be called before using other font-related functions.

Note
SDLXX_core does not have to be initialized before this call.

static sdlxx::core::Version sdlxx::ttf::SDLXX_ttf::getCompiledSdlVersion ( )
static

Get the SDL_ttf version the library was compiled against.

This is determined by what header the compiler used. Note that if you dynamically linked the library, you might have a slightly newer or older version at runtime. That version can be determined with SDLXX_ttf::getLinkedSdlVersion()

Returns
Version Version of SDL_ttf the library was compiled against

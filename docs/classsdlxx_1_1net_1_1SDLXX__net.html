static sdlxx::core::Version sdlxx::net::SDLXX_net::getCompiledSdlVersion ( )

Get the SDL_net version the library was compiled against.

This is determined by what header the compiler used. Note that if you dynamically linked the library, you might have a slightly newer or older version at runtime. That version can be determined with SDLXX_net::getLinkedSdlVersion()

Returns
Version Version of SDL_net the library was compiled against

Open a font file and create a Font of the specified point size.

Some .fon fonts will have several sizes embedded in the file, so the point size becomes the index of choosing which size to load. If the value is too high, the last indexed size will be the default.

Parameters
filePath to the *.ttf or *.fon file
point_sizePoint size (based on 72 DPI) to load
indexIndex of font face from a file containing multiple font faces
